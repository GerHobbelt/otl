OTL 4.0, Examples for PostgreSQL / ODBC

The examples below work with PostgreSQL via ODBC. In general, all of these examples should work with any PostgreSQL 8.0 and higher, when ODBC  is enabled in OTL by setting #define OTL_ODBC, or #define OTL_ODBC_POSTGRESQL. For more examples, see the Generic ODBC examples.

  6. Example 145 (otl_stream::describe_select() with a simple Select)
  7. Example 146 (otl_stream and NULLs)
