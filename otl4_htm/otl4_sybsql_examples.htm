OTL 4.0, Examples for Sybase SQL Server / ODBC

The examples below work with Sybase SQL Server via ODBC. The examples were tested with Sybase SQL Server 15.0 against Sybase stock ODBC drivers. The underlying database had "allow NULLs by default", and "ddl in transactions" enabled. Sybase stock ODBC drivers have a limit of 32767 bytes on SELECTing TEXT and IMAGE columns. So, if you want to be able to fetch longer TEXT and IMAGE values from the database, you may want to try to use ODBC drivers from EasySoft or DataDirect.
