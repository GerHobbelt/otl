    How to file a bug report

    If you have found a problem or a bug in OTL 4.0, feel free to send an email to skuchin@gmail.com, skuchin@gmail.com. If possible, try to put together a self-contained example that reproduces the problem, so I (Sergei Kuchin) can try it on my system and fix the bug quicker.

    If you have a question, look through the F.A.Q. first.
